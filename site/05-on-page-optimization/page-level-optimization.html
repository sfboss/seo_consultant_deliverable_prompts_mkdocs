Page-Level Optimization Prompts

Meta Title and Description Optimization

Purpose

Increase SERP CTR while preserving relevance.

Client Value

More clicks from existing rankings without new content production.

Prompt


Optimize titles and meta descriptions for top [NUMBER_OF_PAGES] pages on [WEBSITE_URL].
For each page include current issues, improved title/description, target keyword, and CTR rationale.

Expected Output Format

Before/after metadata table.

Header Tag (H1-H6) Structure

Purpose

Improve semantic structure and intent matching.

Prompt


Review heading structure for [PAGE_URLS].
Recommend H1-H6 revisions that improve clarity, keyword alignment, and scannability.

Image Optimization (Alt Text, Filenames, Compression)

Prompt


Generate image SEO recommendations for [PAGE_URLS]: filenames, alt text quality, compression opportunities, and next-gen formats.

URL Structure Recommendations

Prompt


Audit URL structures on [WEBSITE_URL].
Provide SEO-friendly URL conventions, canonicalization notes, and redirect mapping considerations.

Internal Linking Suggestions

Prompt


Create internal linking suggestions for [WEBSITE_URL] with source URL, target URL, anchor text, intent, and priority.
