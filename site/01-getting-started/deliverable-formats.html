Understanding Deliverable Formats

Use these standardized formats to keep outputs client-ready.

Recommended format blocks

1. **Executive Summary (5-10 bullets)**

2. **Current State / Baseline**

3. **Opportunity Areas**

4. **Prioritized Recommendations**

5. **Implementation Plan (30/60/90 day)**

6. **KPIs and Reporting Cadence**

7. **Risks, Dependencies, and Assumptions**

Output format request snippet


Return the response as:
1) Executive summary
2) Findings table (Issue, Impact, Evidence, Priority)
3) Action plan table (Task, Owner, Effort, ETA)
4) KPI dashboard definitions
5) Stakeholder-ready next steps
