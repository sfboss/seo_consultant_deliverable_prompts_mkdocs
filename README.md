# SEO Deliverable Prompt Library (MkDocs)

This repository contains a complete MkDocs knowledge base for SEO professionals who need reusable prompts to generate client-ready deliverables across the full SEO lifecycle.

## Features

- Material for MkDocs theme
- Dark/light mode toggle
- Full-text search
- Navigation tabs by SEO phase
- Code copy buttons for prompt blocks
- Tags plugin support for prompt categorization

## Project Structure

- `mkdocs.yml` — site configuration and navigation
- `docs/` — all knowledge base pages
  - `01-getting-started/`
  - `02-discovery-research/`
  - `03-strategy-planning/`
  - `04-technical-seo/`

  - `10-specialized/`
  - `contributing.md`

## Setup

1. Create and activate a virtual environment (recommended).
2. Install dependencies:

```bash
pip install mkdocs mkdocs-material
```

3. Start local server:

```bash
mkdocs serve
```

4. Build static site:

```bash
mkdocs build
```

## Usage

1. Open the homepage and follow the quick-start workflow.
2. Navigate phase-by-phase as you onboard clients.
3. Copy prompts, replace placeholders, and generate deliverables.
4. Use follow-up prompts to deepen analysis and produce implementation-ready plans.
