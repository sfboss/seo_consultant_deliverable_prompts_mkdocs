# Keyword Research Prompts

## Primary + Long-tail + Semantic Cluster Research

### Purpose
Generate a full keyword universe spanning commercial, informational, and navigational intent.

### Client Value
Enables precise targeting of terms most likely to drive qualified traffic and revenue.

### Prompt
```text
Create a keyword research deliverable for [WEBSITE_URL] in [INDUSTRY].

Include sections:
1) Primary commercial-intent keywords (high volume)
2) Long-tail opportunities by problem/use case
3) Semantic keyword clusters by topic entity
4) Keyword difficulty vs opportunity scoring model
5) Local SEO keyword variants for [GEOGRAPHIC_LOCATION]
6) Question-based keywords for PAA/FAQ capture

For each keyword include: intent, estimated volume tier, competition tier, funnel stage, priority score, recommended page type.
```

### Expected Output Format
Master keyword table + cluster map + implementation recommendations.

### Example Usage
`[INDUSTRY]=B2B payroll software`, `[GEOGRAPHIC_LOCATION]=United States`.

### Follow-up Prompts
- "Map this keyword set to existing pages and identify cannibalization risks."
- "Create 20 title/H1 options for the top 5 clusters."

### Tips for Best Results
Ask for confidence levels and assumptions where external tools are not available.
