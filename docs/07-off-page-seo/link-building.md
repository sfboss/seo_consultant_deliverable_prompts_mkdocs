# Link Building Strategy Prompts

## Opportunity Identification

### Prompt
```text
Identify link building opportunities for [WEBSITE_URL] in [INDUSTRY].
Categorize by digital PR, resource pages, partnerships, directories, niche communities, and competitor link gaps.
```

## Outreach Email Templates

### Prompt
```text
Create outreach email templates for link building campaigns:
1) Guest post pitch
2) Resource page inclusion
3) Broken link replacement
4) Digital PR collaboration
Tailor tone to [BRAND_VOICE] and [TARGET_AUDIENCE].
```

## Guest Post Research + Digital PR Angles

### Prompt
```text
Research guest post opportunities and digital PR story angles for [WEBSITE_URL].
Provide relevance scoring, estimated authority tier, outreach notes, and editorial fit rationale.
```

## Local Citation + Broken Link Building Plan

### Prompt
```text
Develop a plan for local citation building and broken-link outreach for [WEBSITE_URL] in [GEOGRAPHIC_LOCATION].
Include prospecting workflow, qualification criteria, outreach cadence, and reporting KPIs.
```
