# Content Brief Generation Prompts

## SEO-Optimized Blog Post Briefs

### Prompt
```text
Create an SEO blog brief for keyword cluster [KEYWORD_CLUSTER].
Include search intent, target persona, SERP analysis, outline, PAA questions, internal links, external references, CTA strategy, and quality checklist.
```

## Landing Page Brief

### Prompt
```text
Generate a landing page SEO brief for [SERVICE_OR_PRODUCT] targeting [TARGET_KEYWORD] in [GEOGRAPHIC_LOCATION].
Include conversion goals, trust elements, section-by-section guidance, schema recommendations, and CRO notes.
```

## Product/Service Page Optimization Brief

### Prompt
```text
Build an optimization brief for [PAGE_URL] with semantic entities, required subsections, FAQs, comparison content, and conversion-focused copy recommendations.
```

## Topic Cluster Planning Brief

### Prompt
```text
Plan a topic cluster for [PILLAR_TOPIC] with one pillar page and [N] supporting articles.
Provide interlinking map, publishing sequence, and KPI expectations.
```
