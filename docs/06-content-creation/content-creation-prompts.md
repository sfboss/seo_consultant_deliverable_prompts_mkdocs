# Content Creation Prompts

## SEO-Optimized Article Generation

### Prompt
```text
Write an SEO-optimized article for [TARGET_KEYWORD] tailored to [TARGET_AUDIENCE].
Requirements:
- Demonstrate first-hand expertise where possible
- Include clear structure with H2/H3 hierarchy
- Integrate semantic variations naturally
- Add FAQ section and schema-ready Q&A
- Include internal link placeholders and conversion CTA
```

## FAQ Section Creation

### Prompt
```text
Generate an FAQ section for [PAGE_TOPIC] focused on high-intent questions from [TARGET_AUDIENCE] in [GEOGRAPHIC_LOCATION].
Return concise, trust-building answers with schema-ready formatting.
```

## Product Description Optimization

### Prompt
```text
Optimize product descriptions for [PRODUCT_CATEGORY] to improve search visibility and conversion.
Include benefit-driven copy, differentiators, semantic keywords, and objection handling.
```

## Local Landing Page Content

### Prompt
```text
Create local landing page content for [SERVICE] in [CITY/REGION].
Include local proof points, service area specificity, trust signals, FAQs, and location-specific intent alignment.
```
