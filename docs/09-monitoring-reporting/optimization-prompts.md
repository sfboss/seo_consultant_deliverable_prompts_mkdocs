# Optimization Prompts

## Underperforming Content Analysis

### Prompt
```text
Identify underperforming pages on [WEBSITE_URL] and diagnose root causes (intent mismatch, weak topical coverage, poor UX, thin authority signals).
Provide page-specific optimization actions.
```

## Ranking Drop Investigation

### Prompt
```text
Investigate ranking drops for [WEBSITE_URL] over [DATE_RANGE].
Assess algorithm volatility, technical changes, content updates, backlink shifts, and SERP competition changes.
Return a hypothesis-ranked root cause analysis.
```

## Algorithm Update Impact Assessment

### Prompt
```text
Assess how recent Google algorithm updates may impact [WEBSITE_URL] in [INDUSTRY].
Map risk areas and produce a mitigation + recovery plan.
```
