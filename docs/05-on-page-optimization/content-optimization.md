# Content Optimization Prompts

## Existing Content Optimization Audit

### Prompt
```text
Audit existing content on [WEBSITE_URL] and identify pages to optimize first based on traffic potential, ranking positions, and conversion value.
Return a prioritized optimization backlog.
```

## Content Refresh Recommendations

### Prompt
```text
For [LIST_OF_URLS], recommend content refresh actions: outdated sections, missing subtopics, weak E-E-A-T signals, thin sections, outdated stats, and schema enhancements.
```

## E-E-A-T Improvement Strategy

### Prompt
```text
Develop an E-E-A-T enhancement plan for [WEBSITE_URL], including author credibility, sourcing quality, trust pages, editorial standards, and review workflows.
```

## Featured Snippet Optimization

### Prompt
```text
Identify snippet opportunities for [WEBSITE_URL] and recommend formatting patterns (definitions, lists, tables, Q&A blocks) per keyword cluster.
```
