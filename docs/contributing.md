# Contributing Guide

Thanks for helping expand this SEO prompt library.

## Contribution principles

- Keep prompts copy-paste ready.
- Preserve the standard format:
  - Purpose
  - Client Value
  - Prompt
  - Expected Output Format
  - Example Usage
  - Follow-up Prompts
  - Tips for Best Results
- Use placeholders to maintain reusability.
- Add tags (`beginner`, `advanced`, `technical`, `creative`).
- Reflect current (2024-2025) SEO best practices.

## Content quality checklist

1. Is the deliverable client-ready?
2. Are recommendations actionable and prioritized?
3. Are assumptions clearly stated?
4. Is the prompt industry-agnostic with easy customization?
5. Does it include measurable outcomes or KPIs?

## How to add a new prompt page

1. Add a new markdown file under the relevant phase directory.
2. Insert the page into `mkdocs.yml` navigation.
3. Validate markdown formatting and heading levels.
4. Run local build checks before opening a PR.
