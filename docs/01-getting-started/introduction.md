# Introduction to Using This Prompt Library

This library is built for agencies, in-house SEO teams, and consultants who need fast, high-quality deliverables.

## Workflow by project stage

1. **Discovery**: Establish baseline performance and opportunities.
2. **Strategy**: Turn findings into a prioritized roadmap.
3. **Execution**: Deploy technical, on-page, content, and off-page initiatives.
4. **Measurement**: Track impact and continuously optimize.

## Prompt execution best practices

- Provide accurate context: business model, target audience, top competitors, and geography.
- Ask the AI to cite assumptions and confidence levels.
- Request output in sections: Executive Summary, Findings, Recommendations, Next Actions.
- Keep an internal QA checklist to verify facts before client delivery.

## Required placeholder set

Use these placeholders in every prompt unless not applicable:

- `[WEBSITE_URL]`
- `[INDUSTRY]`
- `[TARGET_AUDIENCE]`
- `[GEOGRAPHIC_LOCATION]`
- `[COMPETITOR_URLS]`
- `[PRIMARY_GOALS]`
- `[ANALYTICS_NOTES]`
